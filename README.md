# VeChain-Price-Prediction

Предсказание минимальной и максимальной цен криптовалюты VeChain на одну свечу вперед при помощи:
--- Линейной регресси
--- Случайного леса
--- Градиентного бустинга - CatBoost (выбран по умолчанию)
